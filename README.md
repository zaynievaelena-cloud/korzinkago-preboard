# Korzinka Go Preboarding Landing

Готовая папка для публикации через GitHub Pages.

## Как загрузить

1. Создайте новый публичный репозиторий на GitHub, например `korzinkago-preboard`.
2. Загрузите **всё содержимое этой папки** в корень репозитория.
3. Проверьте, что `index.html` лежит в корне репозитория, а не внутри дополнительной папки.
4. Откройте `Settings → Pages`.
5. Выберите `Deploy from a branch`.
6. Branch: `main`, Folder: `/root`.
7. Нажмите `Save`.

После публикации ссылка будет в формате:

`https://ВАШ-ЛОГИН.github.io/korzinkago-preboard/`

Для проверки с телефона сначала откройте:

`https://ВАШ-ЛОГИН.github.io/korzinkago-preboard/healthcheck.html`
